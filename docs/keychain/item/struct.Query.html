Query for keychain items with the provided SecAttrApplicationLabel (not to be confused with a SecAttrLabel), i.e. the hash/fingerprint of a public key in the keychain.

Both the private and public key in a keypair have a SecAttrApplicationLabel set to the public key's fingerprint.
